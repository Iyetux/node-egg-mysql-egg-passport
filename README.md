## 这是一个完整的前后端项目，此为后端部分，前端部分由react和Antd实现

前端在仓库的另一个项目里面

## Installation

数据库mysql,egg-passport, 请先查看config/config_default.js修改配置

```bash
$ npm install
$ npm run dev
$ open http://localhost:7001/
```

## 功能介绍

- Use `登录注册.
- Use `用户管理：增删改查，修改权限.
- Use `操作日志记录.
- Use `消息推送功能.
- Use `图片上传功能.
- Use `Ipc demo.
